# 🌍 Climate Change Data Storytelling Dashboard

An interactive data analytics dashboard built using Python, Streamlit, and Plotly to analyze and visualize climate change trends through storytelling.

## 📌 Project Overview
This project explores:
- Global temperature rise
- CO₂ concentration growth
- Sea level rise
- Correlation between emissions and warming
- Future temperature prediction using Machine Learning

The dashboard transforms raw climate data into meaningful insights through advanced visualizations.

---

## 🚀 Features
✔ Interactive dashboard  
✔ Advanced Plotly visualizations  
✔ Climate impact bubble charts  
✔ CO₂ vs Temperature trend analysis

## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/yourusername/climate-change-dashboard.git
cd climate-change-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```
